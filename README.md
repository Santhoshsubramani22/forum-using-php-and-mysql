
# 🗣 Forum Using PHP and MySQL

A simple discussion forum web application built using PHP and MySQL.

---

## 🚀 Features

- Home page displays all available topics
- Users can create new topics
- View discussion threads and posts
- Vote or interact with topics
- Styled using CSS

---

## 🧱 Requirements

✔ XAMPP or any local server with  
  • PHP support  
  • MySQL database  

---

## 🗂 Installation

1. Clone or download the repository:
   ```bash
   git clone https://github.com/Santhoshsubramani22/forum-using-php-and-mysql.git
````

2. Open **XAMPP** and start **Apache** & **MySQL**.
3. Import the database:

   * Open **phpMyAdmin**
   * Click **Import**
   * Choose `forum.sql` file from the project
4. Copy the project folder into `htdocs` (e.g., `C:\xampp\htdocs\forum`)
5. In your browser, visit:

   ```
   http://localhost/forum
   ```

---

## 📁 Project Files

* `index.php` — Forum homepage
* `create_topic.php` — Create a new topic
* `topic.php` — View topic and comments
* `vote.php` — Voting logic
* `config.php` — Database connection
* `style.css` — Stylesheet
* `forum.sql` — SQL database file

---

## 📌 How to Use

1. Open your browser and go to:

   ```
   http://localhost/forum
   ```
2. Add a discussion topic.
3. Click any topic to view replies.
4. Interact with threads and explore site features.
